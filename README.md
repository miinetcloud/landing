# MiinetCloud Landing Page

A brutalist temporary landing page for MiinetCloud.

## Features

- **Brutalist Design**: Bold, bulky fonts with thick borders and high contrast
- **Primary Color**: #003088 (MiinetCloud blue)
- **White Logo**: Placeholder logo in white (replace `logo.svg` with your actual logo)
- **Login Button**: For staff access (links to `/login`)
- **GitHub Link**: Links to [github.com/miinetcloud](https://github.com/miinetcloud)

## Usage

Simply open `index.html` in a browser or deploy to your web server.

Replace `logo.svg` with your actual white logo file (supports PNG, SVG, or other image formats).
