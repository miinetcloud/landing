# MiinetCloud Landing Page

A Carhartt-style brutalist landing page built with Svelte and bold animations.

## Features

- **Svelte Framework**: Modern, reactive component framework
- **Carhartt-Style Brutalism**: Utilitarian design with bold typography and industrial aesthetics
- **Bold Animations**: Page load animations, hover interactions, and dynamic effects
- **Primary Color**: #003088 (MiinetCloud blue)
- **White Logo**: Loaded from https://obj.miinet.cloud/branding/Wit.svg
- **Login Button**: For staff access (links to `/login`)
- **GitHub Link**: Links to [github.com/miinetcloud](https://github.com/miinetcloud)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Build the project with `npm run build` and deploy the contents of the `dist` directory to your web server.
